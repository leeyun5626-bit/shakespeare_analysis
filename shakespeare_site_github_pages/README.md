# Shakespeare Narrative Types Website

이 폴더는 GitHub Pages에 바로 올릴 수 있는 정적 웹사이트입니다.

## 파일 구성
- `index.html`
- `style.css`
- `script.js`
- `assets/images/` (업로드하신 27개 이미지)

## GitHub Pages 배포 방법
1. GitHub에서 새 저장소(repository)를 생성합니다.
2. 이 폴더 안의 파일을 모두 업로드합니다.
3. 저장소의 **Settings → Pages**로 이동합니다.
4. **Build and deployment**에서 Source를 **Deploy from a branch**로 선택합니다.
5. Branch를 `main` / root 로 설정하고 저장합니다.
6. 잠시 후 GitHub Pages 주소가 생성됩니다.

## 참고
- 이미지 경로는 상대경로로 연결되어 있어 그대로 업로드하면 됩니다.
- 첫 화면은 `index.html`입니다.
